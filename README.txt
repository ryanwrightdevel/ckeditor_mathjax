Mathematical Formulas
=====================

Description
===========
Mathematical Formulas plugin that introduces the MathJax widget. You can use it
to create or modify equations using TeX.

Usage
=====
Go to the Text formats and editors settings (/admin/config/content/formats) and
add the Mathematical Formulas Button to any CKEditor-enabled text format you
want.

Installation
============
1. Download the plugin from http://ckeditor.com/addon/mathjax
2. Place the plugin in the root libraries folder (/libraries).
3. Enable CKEditor Mathematical Formulas module in the Drupal admin.

Each filter format will now have a config tab for this plugin.

Dependencies
============
This module requires:

* CKEditor (core)

Uninstallation
==============
1. Uninstall the module from 'Administer >> Modules'.
